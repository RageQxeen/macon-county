# Macon County Website Redesign

## Overview

This project is a complete redesign of the Macon County, North Carolina website. The goal of this redesign is to create a modern, user-centered, and responsive website that improves accessibility, navigation, and overall usability for residents, visitors, and county staff.

The site was built as part of the Web Design course (WEB 210) and follows all assignment requirements, including responsive design, accessibility standards, and a structured multi-page layout.

---

## Project Goals

* Improve navigation clarity and reduce user frustration
* Provide a fully responsive layout for all screen sizes
* Create a clean, professional government-style design
* Ensure accessibility using WCAG guidelines
* Optimize performance for users with slower internet connections
* Organize content into clear, easy-to-find sections

---

## Features

* Responsive design using Bootstrap
* Mobile-friendly navigation with two-level dropdown menus
* Consistent template across all pages
* 7 fully connected pages
* Accessible structure with semantic HTML
* Optimized images and lightweight performance
* SEO-friendly meta tags

## Accessibility

This project follows WCAG accessibility guidelines by including:

* Semantic HTML structure
* Proper heading hierarchy
* Alt text for images
* High contrast color choices
* Keyboard-friendly navigation
* Responsive text and layout

The site was tested using the WAVE accessibility evaluation tool.

---

## Performance Optimization

* Compressed and optimized images
* Minimal use of external resources
* Clean and efficient CSS
* Mobile-first responsive design

## Installation / Usage

To run this project locally:

1. Download or clone the repository

3. Open `index.html` in a web browser
